\section{Reproduction Commands}
\label{app:reproducibility}

Run the fast Python smoke path from the repository root:
{\scriptsize
\begin{verbatim}
python scripts/experiments/run_full_regression.py
python scripts/experiments/run_final_ndss_regression.py
\end{verbatim}
}

Run full SP1 proof refresh on Linux, WSL2 Ubuntu, or Kaggle with SP1 installed:
{\tiny
\begin{verbatim}
python3 scripts/experiments/benchmark_distinct_td_sp1.py --prove
python3 scripts/experiments/benchmark_forward_td_mlp_sp1.py --prove
python3 scripts/experiments/benchmark_mountaincar_forward_td_sp1.py --prove
python3 scripts/experiments/benchmark_one_step_sgd_tiny_sp1.py --prove
python3 scripts/experiments/run_final_ndss_regression.py
\end{verbatim}
}

If the distinct-TD proof run is unstable, prove accepted cases separately:
{\tiny
\begin{verbatim}
python3 scripts/experiments/benchmark_distinct_td_sp1.py --prove --prove-cases TD-1
python3 scripts/experiments/benchmark_distinct_td_sp1.py --prove --prove-cases TD-2
python3 scripts/experiments/benchmark_distinct_td_sp1.py --prove --prove-cases TD-4
python3 scripts/experiments/benchmark_distinct_td_sp1.py --prove --prove-cases TD-8
\end{verbatim}
}

\section{Tamper-Test Definitions}
\label{app:tamper-tests}

The final tamper matrix includes distinct-batch attacks on duplicate index,
wrong item index, swapped item order, wrong item loss, wrong claimed batch
average, and wrong path order. It includes forward-TD attacks on online model
weight, target model weight, activation, ReLU mask, argmax, selected target
value, and claimed batch loss. It includes tiny-update attacks on gradient
tensor, delta tensor, learning rate, post model weight, post model commitment,
and SmoothL1 gradient. The broader regression suite also retains legacy TD MVP
tamper tests for schema, reward, done, rounding, Merkle, target, and TD-error
cases.

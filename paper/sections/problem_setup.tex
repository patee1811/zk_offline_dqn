We consider a discounted Markov decision process with state space $\states$,
action space $\actions$, reward $r$, terminal flag $d$, and discount factor
$\gamma \in [0,1)$. The offline dataset is fixed before verification:
\[
\dataset = \{(s_i,a_i,r_i,s'_i,d_i)\}_{i=1}^N .
\]
Each transition is serialized into fixed-point integers and committed in a
SHA-256 Merkle tree~\cite{merkle1987}. The public dataset commitment is the
Merkle root. A membership witness consists of the private transition, canonical
leaf encoding, leaf hash, leaf index, and sibling path.

For a DQN with online parameters $\theta$ and target parameters $\bar{\theta}$,
the Double-DQN target for transition $i$ is
\begin{align}
a_i^* &= \arg\max_{a' \in \actions} Q_\theta(s'_i,a'), \\
y_i &= r_i + \gamma (1-d_i) Q_{\bar{\theta}}(s'_i,a_i^*).
\end{align}
The TD error and SmoothL1 loss are
\[
\delta_i = Q_\theta(s_i,a_i) - y_i,
\qquad
\ell_i =
\begin{cases}
\frac{1}{2}\delta_i^2 & \text{if } |\delta_i| < 1,\\
|\delta_i|-\frac{1}{2} & \text{otherwise.}
\end{cases}
\]
For a minibatch $B$, the claimed batch loss is the integer average of per-item
fixed-point losses.

The implementation uses scale $S=1000$ and deterministic integer arithmetic.
The TD-only relation treats the Q-values as private witness values and proves
membership plus Bellman/TD/loss consistency. The forward-TD relation additionally
hashes quantized model weights to public model commitments and checks MLP
forward passes, ReLU masks, online argmax, target-network selection, Bellman
target, and loss. The tiny SGD relation extends this with a single
fixed-point-gradient update for a small network.

\subsection{Python Regression and Smoke Path}

The full Python regression currently contains 15 checks and passes 15/15. It
compiles Python modules, verifies minibatch artifacts, checks forward
consistency, verifies one-step and short-trace artifacts, runs negative suites,
and executes Python-only benchmark smoke tests. The smoke tests now write to
separate \path{*_python_smoke} directories so that they do not overwrite
SP1 proof summaries used by the paper.

\subsection{Interpretation}

The final artifact supports three classes of claims. First, Python semantic
oracles check distinct replay minibatch TD, model-grounded forward-TD MLP, and
tiny one-step SGD fixtures with tamper rejection. Second, report generation
surfaces existing benchmark and tamper matrices with provenance. Third, SP1
proof generation and verification are validated for the TD MVP canonical vector
only. These are relation-level results, not a full proof of training.

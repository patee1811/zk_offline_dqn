\subsection{Python Regression and Smoke Path}

The full Python regression currently contains 14 checks and passes 14/14. It
compiles Python modules, verifies minibatch artifacts, checks forward
consistency, verifies one-step and short-trace artifacts, runs negative suites,
and executes Python-only benchmark smoke tests. The smoke tests now write to
separate \path{*_python_smoke} directories so that they do not overwrite locked
SP1 proof summaries used by the paper.

\subsection{Interpretation}

The final artifact supports three claims. First, distinct replay minibatch TD is
proved in SP1 for CartPole batch sizes 1, 2, 4, and 8. Second, model-grounded
forward-TD is proved in SP1 for both CartPole and a second-environment
MountainCar fixture. Third, a micro-scale one-step SGD update is proved in SP1
for a CartPole 4-8-2 network. These are relation-level proofs, not a full
proof of training.
